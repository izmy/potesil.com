# potesil.com

Personal website of Tomáš Potěšil, built with [Astro](https://astro.build/).

## 🛠️ Usage

Install dependencies:

```sh
npm install
```

Start development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```
